URL Shortener

¿en que consiste?

URL Shortener es una aplicación web que permite transformar enlaces largos en versiones cortas y fáciles de compartir. Desarrollada con HTML, Tailwind CSS y JavaScript puro en el frontend, y Node.js con Express en el backend, esta herramienta demuestra habilidades clave como diseño responsivo, manejo de rutas, y lógica de servidor. Ideal para quienes buscan eficiencia, simplicidad y un ejemplo real de desarrollo full stack.
